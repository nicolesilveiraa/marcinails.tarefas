Sistema de Lista de Tarefas - Nail Manager 
Projeto focado em DevOps e Gerência de Configuração.

Itens de Configuração (CI)
**Código Fonte:** `/src` (HTML e CSS separados).
**Scripts:** `/scripts/validar_projeto.sh`.
**Pipeline:** `.github/workflows/main.yml`.
**Interface:** Protótipos desenvolvidos no Figma.

Gerência de Configuração
Utilizamos o Git para versionamento atômico, garantindo que cada alteração no código ou na documentação seja registrada e validada automaticamente pelo Pipeline de CI.
